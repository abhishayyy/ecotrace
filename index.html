Create a Free Firebase Project

EcoTrace uses Firebase for secure user authentication and cloud syncing across your devices.

  1. Go to the Firebase Console.
  2. Click Add Project, name it "EcoTrace", and click through to create it.
  3. Once loaded, select the Web App Icon (</>) on the dashboard to register a new web application.

Enable Authentication & Firestore

Activate the required serverless features inside your Firebase project dashboard:

  • Authentication: Go to Build → Authentication, click "Get Started", enable the Email/Password provider.
  • Firestore: Go to Build → Firestore Database, click "Create Database", select "Start in Test Mode" (allows quick testing), and complete setup.
